# Project 1

# Smart Campus Dashboard

## A simple and interactive student productivity dashboard built using HTML, CSS, and JavaScript.

This project helps students manage their daily academic activities in one place, such as assignments, notes, timetable, and real-time updates.

---

# Project Overview

## The Smart Campus Dashboard is a single-page web application designed to make academic management easier and more organized.

It provides useful tools for students to keep track of assignments, class schedules, notes, and important daily information.

# The goal of this project is to create a dashboard that is:

* Easy to use
* Visually clean
* Interactive
* Useful for students in day-to-day campus life

---

# Core Features

## Add Assignments

* Add assignment title and deadline

## Display Assignments

* View all saved assignments in a list

## Delete Assignments

* Remove assignments when no longer needed

## Timetable Section

* Display class schedule

## Notes Section

* Add and save personal academic notes

## Real-Time Digital Clock

* Live clock updates automatically

## Current Date Display

* Shows the current date along with time

---

# Extra / Custom Features

## 1. Home Dashboard

A summary dashboard page that gives a quick overview of:

* Greeting message along with your CPI display
* Current date and time
* Assignments due on that day

## 2. Today’s Lectures Section

A dedicated section on the home dashboard displays the lectures/classes scheduled for the current day.

## 3. Motivational Quote Box (API-Based)

A motivational quote section is included in the dashboard to make the interface more engaging and inspiring for students.

This feature helped in learning:

* Fetch API
* Async JavaScript
* Working with external APIs
* Displaying dynamic content on the webpage

---

# Technologies Used

* HTML – Structure of the webpage
* CSS – Styling and layout
* JavaScript – Functionality and interactivity
* LocalStorage – Data persistence in browser
* Fetch API – For retrieving motivational quotes from an external API
